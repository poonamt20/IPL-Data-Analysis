# IPL Data Analysis using Python

This project performs Exploratory Data Analysis (EDA) on IPL dataset using Python.

## 📊 Features
- Most winning teams analysis
- Top batsmen and bowlers
- Strike rate calculation
- Toss impact analysis
- Venue-based analysis

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

## 📁 Dataset
IPL match and ball-by-ball data

## 📌 Key Insights
- Identified top performing teams and players
- Found impact of toss on match outcomes
- Analyzed player performance using strike rate
